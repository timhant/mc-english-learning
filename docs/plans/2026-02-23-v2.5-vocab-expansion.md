# v2.5 词库扩充设计文档
*日期：2026-02-23*
*状态：已确认*

## 概述
将词库从 156 词扩充至 ~500 词，同时改造分级机制为"游戏进程分级 + 超纲可触发"。

## 核心改动

### 1. 分级机制改造

**当前等级词汇**触发完整体验：
- 大标题显示（Title）
- 英文 + 中文语音朗读
- 音标显示
- 首次发现"新词解锁!"庆祝（粒子 + 音效 + XP）
- 计入进度，影响升级

**超纲词汇（未解锁等级）**触发轻量体验：
- ActionBar 小字显示（英文 + 中文）
- 英文 + 中文语音朗读（保留）
- 无庆祝特效、无 XP
- 不计入进度
- 冷却时间加倍（120 秒 vs 60 秒）

## 3. 代码改动清单

### 3.1 新文件结构
```
vocab/
  index.js          # 汇总 + 按等级构建 Map + 新增 getWordType() 方法
  level1.js ~ level5.js  # 重写，按游戏进程分级，每级 ~100 词
```

### 3.2 proximity.js 改动
- 检测到实体时，判断是否在当前等级
- 当前等级 → 调用 fullExperience()
- 超纲 → 调用 lightExperience()

### 3.3 blockLearn.js 改动
- 同上逻辑

### 3.4 新增 lightExperience 函数
- ActionBar 显示：`§7{EN} §8| §7{CN}`（灰色，低调）
- 语音照常播放
- 无 Title、无粒子、无 XP
- 冷却 120 秒

### 3.5 progress.js 改动
- 只追踪当前等级的词
- 超纲词不计入升级进度
- 但可以记录"已预览"状态（为"重逢惊喜"做准备）

### 3.6 voice.js
- 无需改动（已修复坐标问题）
- 新词只需在 sound_definitions.json 注册 + 提供 ogg 文件

### 3.7 sound_definitions.json + 音频文件
- 新增约 344 个词 × 2 = 688 个新 ogg 文件
- 使用 Edge TTS 生成（mc-tts-audio skill）

### 3.8 config.json
- 新增 `outOfLevelCooldown: 120`
- 新增 `outOfLevelEnabled: true`（可关闭超纲触发）
